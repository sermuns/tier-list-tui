## Installation

For each version, prebuilt binaries are automatically built for Linux, MacOS and Windows.

- You can download and unpack the
  latest release from the [releases page](https://github.com/sermuns/tier-list-tui/releases/latest).

- Using [`cargo-binstall`](https://github.com/cargo-bins/cargo-binstall)

  ```sh
  cargo binstall tier-list-tui
  ```

- From source with Cargo

  ```sh
  cargo install tier-list-tui
  ```
